tail -f 1.txt|grep -i err

tail 1.txt - вывод 10 последних строк
-f - отслеживание изменений
-n* - нербходимое количество строк

grep cat 1.txt - поиск по файлу
grep -R cat . (поиск по файлам начиная с текущей директории)
-R (поиск по файлам в папке) grep -R cat dir1/
-i (регистронезависимый)
-v (вывести несовпадающие строки)
-с (подсчитать строки где встречается)
